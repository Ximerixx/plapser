# Инструкция по установке PWA на телефон

## Требования

1. **HTTPS соединение** - PWA работает только через HTTPS (или localhost для разработки)
2. **Поддерживаемый браузер** - Chrome, Firefox, Safari, Edge

## Как установить на Android

### Chrome (Android)

1. Откройте сайт в Chrome
2. Дождитесь появления баннера "Добавить на главный экран" внизу экрана
3. Или нажмите меню (три точки) → "Добавить на главный экран"
4. Подтвердите установку
5. Приложение появится на главном экране

**Если баннер не появляется:**
- Проверьте, что сайт открыт через HTTPS
- Откройте DevTools (F12) → Application → Service Workers - проверьте, что Service Worker зарегистрирован
- Откройте Application → Manifest - проверьте, что manifest.json загружается без ошибок
- Очистите кэш браузера и перезагрузите страницу

### Firefox (Android)

1. Откройте сайт в Firefox
2. Нажмите меню (три точки) → "Страница" → "Установить"
3. Или используйте кнопку в адресной строке (если доступна)
4. Подтвердите установку

## Как установить на iOS (iPhone/iPad)

### Safari (iOS)

1. Откройте сайт в Safari
2. Нажмите кнопку "Поделиться" (квадрат со стрелкой вверх)
3. Прокрутите вниз и выберите "На экран «Домой»"
4. Нажмите "Добавить"
5. Приложение появится на главном экране

## Проверка установки

После установки приложение должно:
- Открываться в отдельном окне (без адресной строки браузера)
- Иметь свою иконку на главном экране
- Работать офлайн (если данные были закэшированы)

## Отладка

Если установка не работает:

1. **Проверьте HTTPS:**
   - Убедитесь, что сайт открыт через `https://`, а не `http://`
   - PWA не работает без HTTPS (кроме localhost)

2. **Проверьте Service Worker:**
   - Откройте DevTools (F12)
   - Перейдите в Application → Service Workers
   - Должен быть зарегистрирован service-worker.js
   - Статус должен быть "activated and is running"

3. **Проверьте Manifest:**
   - Откройте Application → Manifest
   - Должен отображаться manifest.json без ошибок
   - Проверьте, что все поля заполнены

4. **Проверьте консоль:**
   - Откройте Console в DevTools
   - Должно быть сообщение: `[Service Worker] Registered: ...`
   - Не должно быть ошибок

5. **Очистите кэш:**
   - Application → Storage → Clear site data
   - Перезагрузите страницу

## Создание иконок

Если иконки отсутствуют, создайте их:

- `icon-192.png` - 192x192 пикселей
- `icon-512.png` - 512x512 пикселей

Поместите файлы в папку `/public/`

Можно использовать онлайн-генераторы:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
